Favicon Ball - Multiple Sizes

Instructions:

  1. Right-click on a ball and select "Inspect Element"
  2. In DevTools, right-click on the ball element and select "Capture node screenshot"
  3. This will save a PNG with transparent background
  4. Use an online converter like favicon.io or convertio.co to convert PNG to ICO
  5. Replace favicon.ico in your website root with the new file

Note: The ball will have a transparent background when captured using DevTools screenshot.
